Exploitation Phase

The JavaScript was deobfuscated as follows:

function verifyInviteCode(code){
    var formData = {"code": code};
    $.ajax({
        type: "POST",
        dataType: "json",
        data: formData,
        url: '/api/v1/invite/verify',
        success: function(response){
            console.log(response);
        },
        error: function(response){
            console.log(response);
        }
